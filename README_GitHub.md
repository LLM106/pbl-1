# GitHub 업로드용 패키지

이 폴더는 GitHub Pages 업로드용으로 정리된 버전입니다.

## 구성
- `index.html` : 첫 화면 요약 페이지
- `report.html` : 전체 보고서 본문
- `assets/styles.css` : 스타일 파일
- `media/` : 보고서 이미지 파일

## 업로드 방법
1. GitHub에서 새 repository 생성
2. 이 폴더 안의 파일과 폴더를 **그대로 전부** 업로드
3. `Settings > Pages`에서 `main` 브랜치와 `/root` 선택
4. 몇 분 뒤 생성된 주소 접속

## 주의
- `index.html`과 `report.html`은 `media/` 폴더를 참조하므로, 이미지 폴더까지 같이 올려야 페이지가 정상 표시됩니다.
- 파일명은 이미 맞춰져 있으니 바꿀 필요가 없습니다.
