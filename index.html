이 연구가 보는 것

라틴·그리스어 파생 명사의 복수형 변화는 단순히 규칙형이 고전형을 밀어내는 직선적 과정이 아니라는 점을 밝히는 연구다. 보고서는 각 표본을 현재 사용 상태, 변화의 속도·방향, 작동 메커니즘의 세 층위로 분석한다.

표본과 분류

총 11개 명사쌍을 대상으로 분석했으며, 현재 상태 기준으로 규칙형 우세 2쌍, 고전형 우세 6쌍, 공존 3쌍으로 분류한다.

  • 규칙형 우세: formulas/formulae, antennas/antennae
  • 고전형 우세: criteria, curricula, cacti, matrices, premia, syllabi
  • 공존: referendums, aquaria, indexes/indices

핵심 결론

규칙형 확대

대중적·실용적 맥락에서 이미 강한 기반을 확보한 규칙형은 2040년까지 우세를 더 강화할 가능성이 높다.

고전형 선택적 유지

학술적·전문적 공동체 안에서 중심 형태로 기능하는 고전형은 단기간에 사라지기보다 현재 위상을 유지할 가능성이 크다.

기능적 공존

의미 분화나 register 분화가 형성된 표본은 한쪽의 소멸보다 역할 분담을 유지하는 방향으로 공존할 가능성이 높다.

대표 사례

formulas / formulae규칙형 우세 + register 분화
antennas / antennae규칙형 우세 + 의미 분화
criteria / criterions고전형 우세 + 빠른 수렴
indexes / indices공존 + 기능 분담

이 패키지 구성

  • index.html — 요약 페이지
  • report.html — 전체 본문
  • assets/styles.css — 공통 스타일
  • media/ — DOCX에서 추출한 이미지 파일

즉, 예전처럼 40MB짜리 단일 HTML 하나를 올리는 대신, 텍스트·스타일·이미지를 분리해 GitHub 업로드와 수정이 쉬운 구조로 정리한 버전이다.

업로드 순서

  1. 새 GitHub repository를 만든다.
  2. 이 패키지 안의 파일과 폴더를 그대로 업로드한다.
  3. Settings → Pages에서 main / root를 선택한다.
  4. 조금 기다린 뒤 생성된 주소로 접속한다.

파일명은 이미 index.html로 맞춰 두었으니, 따로 이름을 바꿀 필요가 없다.
